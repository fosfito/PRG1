Subir archivo a un branch
=========================

1) git add . //Agregar stage
2) git commit -m "texto" //agrega el stage al repo local
3) git push origin nombre_branch //subirlo a github (Brunch main)

Primer commit 
==============

git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin <url de tu repo>
git push -u origin main
